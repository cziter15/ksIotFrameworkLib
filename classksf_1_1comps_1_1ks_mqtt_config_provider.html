Detailed Description

A component that manages MQTT-related configuration.

This configuration provider handles MQTT properties. Detailed documentation on its behavior can be found on the ksConfigProvider sub-page.

This method reads the configuration file to retrieve MQTT broker information from file. The config contains data like MQTT address, device prefix, optional: credentials and expected SSL fingerprint.
